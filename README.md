🍔 Food Ordering Website

A full-stack food ordering web application where users can explore food categories, select items, manage their cart, apply coupons, and view an attractive dashboard experience similar to modern restaurant platforms.

🚀 Features
🔐 User Authentication & Authorization
🎨 Attractive and Responsive Dashboard UI
🍕 Food Categories Filtering
🥗 Dynamic Food Selection System
🛒 Add to Cart & Remove from Cart
💰 Automatic Total Price Calculation
🎟️ Coupon Discount Functionality
📦 Order Summary Section
🖼️ Dynamic Food Images Rendering
⚡ Fast and Interactive User Experience
🧠 Project Workflow
User Signup/Login using authentication system.
After login, user is redirected to a modern dashboard.
Different food items are displayed with images:
Cake
Salad
Biryani
Pizza
And many more.
User can click on a specific category to view only related food items.
Users can:
Add food items
Remove items
Manage quantity
Cart automatically updates:
Total price
Selected items summary
Users can apply coupon codes for discounts.
Final experience works similar to a real restaurant food ordering platform.
